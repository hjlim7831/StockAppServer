# Stock App &#128200; Backend

## 개발 환경
- Spring Boot 2.7.4
- Spring Tools 4 for Eclipse
- Java 8
- MyBatis
- Gradle

## 주요 구현 내용
- 실시간 주가, 환율 정보 조회
- 관련 뉴스 크롤링
- 주식 및 외환 매매
- 주식 종목 추천 알고리즘
